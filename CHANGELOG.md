# Changelog

## 0.1.0

Initial scaffold release.

### Added

- File, raw Markdown string, stdin, batch, and compile input modes.
- Typer + Rich CLI.
- Pydantic render and logging settings.
- markdown-it-py rendering backend.
- Jinja HTML builder and Pygments highlighting.
- WeasyPrint PDF renderer.
- Optional Playwright, Pandoc, Python-Markdown, PDF, and ultilog extension points.
- Built-in themes and profiles.
- Configuration discovery from `markprint.toml`, `.markprint.toml`, and `pyproject.toml`.
- Unit, integration, e2e, and visual test scaffolds.
